1. SEJARAH PARROT OS
Parrot didasarkan pada cabang pengujian Debian (Bullseye), dengan kernel Linux 5.8. Ini mengikuti model 
pengembangan rilis bergulir.Lingkungan desktop adalah MATE dan KDE , dan manajer tampilan default adalah 
LightDM. Sistem disertifikasi untuk berjalan pada perangkat yang memiliki RAM minimal 256MB , dan cocok 
untuk arsitektur prosesor 32-bit ( i386 ) dan 64-bit (amd64). Selain itu, proyek ini tersedia untuk 
arsitektur ARMv7 ( armhf ).

Pada Juni 2017, Tim Parrot mengumumkan bahwa mereka mempertimbangkan untuk beralih dari Debian ke Devuan, 
terutama karena masalah dengan systemd. Mulai 21 Januari 2019, tim Parrot telah mulai menghentikan pengembangan 
ISO 32-bit ( i386 ) mereka secara bertahap.
Pada Agustus 2020, Parrot OS secara resmi mendukung Lightweight Xfce Desktop. Parrot didasarkan pada cabang 
pengujian Debian (Bullseye), dengan kernel Linux 5.8. Ini mengikuti model pengembangan rilis bergulir.
Lingkungan desktop adalah MATE dan KDE , dan manajer tampilan default adalah LightDM.


2. TENTANG PARROT OS

Parrot Linux adalah distro yang khusus di desain untuk Pentesting, Computer Forensic, Reverse Engineering dan Hacking, 
Cloud Pentesting, Privacy Anonimity, Cryptograhphy. Operating System ini berdasarkan pada Debian dan di Bangun oleh 
Frozen Box Network.
Dalam rilis resminya Parrot ditujukan untuk setiap orang dari mulai amatir sampai dengan Proffesional, karena menyediakan 
tool yang telah dikombinasi sedemikian rupa sehingga mudah digunakan, ringan untuk pen testing serta dapat digunakan untuk 
setiap hari.
Untuk OS Parrot nya bisa di download di http://www.parrotsec.org/ , ukuranya sekitar 1,8 GB, sampai dengan sekarang sudah 
dapat versi 1.9 dengan code name CyberLizard.
